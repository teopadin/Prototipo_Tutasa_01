# Resultado Costo VS Ventas

Archivos para reemplazar/completar la carpeta:

`Prototipos_TUTASA/Administración/Resultado_Costo_VS_Ventas`

El caso queda con:

- Empresa de transporte desplegable.
- Periodo mensual (`MM/yyyy`).
- Botón `Mostrar`.
- Tabla con `Periodo`, `Ventas`, `Costos` y `Resultado`.
- Mensaje si no existen movimientos para la empresa y periodo seleccionados.

La lógica considera:

- Ventas: importes de guías entregadas incluidas en hojas de ruta de transporte de la empresa seleccionada.
- Costos: arrendamientos mensuales de bodega de la empresa seleccionada.
- Resultado: `Ventas - Costos`.
